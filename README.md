Cooling System is a high-fidelity monitoring solution designed for cold chain logistics and warehouse management. It provides real-time visibility into temperature-sensitive environments to prevent spoilage and ensure regulatory compliance.



## üöÄ Key Features

* **Real-Time Telemetry:** Instant feedback on internal temperature and humidity levels.
* **Safety Threshold Logic:** Integrated status indicators that transition to 'Critical' (Red) if temperatures deviate from the $2^{\circ}C$ to $8^{\circ}C$ safety window.
* **Predictive Analysis:** Displays AI-calculated predicted temperatures to anticipate cooling failures.
* **Ambient Outside Tracking:** Live synchronization with Tomorrow.io API to correlate internal fluctuations with external weather conditions.
* **Responsive Industrial UI:** Optimized for warehouse tablets and control room monitors.
